제거 : 디렉토리에서 완전히 제거
폐기 : 변경사항만 롤백

soft reset : 변경사항을 stage에 살려놓은 reset
mixed reset : 변경사항을 stage에 올려놓기 전(밑에)으로 reset
hard reset : 해당 버전으로 아예 싹 지우고 reset

태그를 통해서 버전 태그를 붙여줄 수 있음

스태시에 저장하면(stage올리지 말고?) 변경상태를 저장해주고 현 파일은 변경전으로 돌아감
스태시 적용을 하면 저장되어있던 스태시 내용으로 사용 가능

내것을 이용해 충돌 해결 : 현재 선택되어 있는 브랜치 기준으로 적용
저장소 것을 사용하여 충돌 해결 : master에 있는 브랜치 기준으로 적용
