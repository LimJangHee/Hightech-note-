stdin(standard in) : 표준(기본) 입력 장치 ex)키보드 -> System.in
stdout(standard out) : 표준(기본) 출력 장치 ex)모니터 -> System.out

기본자료형 : 상수값들을 표현하기위한
boolean char int double
참조자료형
String

overflow : 저장 가능한 값을 넘어섬(양의 값으로)(300>256)
underflow : 저장 가능한 값을 넘어섬(음의 값으로)(0>-10)

8bit = 1byte
맨 앞에 있는 비트는 부호 비트
0: 양수, 1: 음수
0 ~ 255 표현 -> unsigned
-128 ~ 127 표현 -> signed(자바는 여기에 해당)

다만 컴퓨터인식하는 마이너스 값은 이랑 다름
즉 3을 -3이라고 표현하려면 보수를 사용하는데 이는 양수의 bit 반전값에다 1을 더한값을 말한다
이 값을 양수를 표현한 bit값에다 넣으면 1000000000 이런식으로 나오는데 맨 앞값을 버리고 0이라고 생각

기본자료형의 변수들은 스택(stack) 메모리에 할당됨
하지만 할당량이 적기 때문에 힙(heap)이 더 많이 사용됨 -> 나중에 배울것임
