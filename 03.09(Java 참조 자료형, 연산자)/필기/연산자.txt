단항연산자 : 부호와 같이 변수 하나의 관계를 지정해주는 것
이항연산자 : 두개의 변수간에 관계를 지정해주는 것(사칙연산 등)

중위표현식 : 3+4*5
후위표현식 : 3 4 5 + * -> 컴퓨터가 해석하는 방식

/ : 몫을 구할때
% : 나머지를 구할때 -> 오직 정수형에서만 사용

대입연산자가 연산자 중에 제일 늦게 적용(떨어져 있는 경우에는 =이 순서대로 적용됨)
복합대입 연산자 : += -= *= /= %=

++a : 전위증가 연산자
a++ : 후위증가 연산자
출력(print)을 예로 들어보면 전위는 증가하고 출력하고 후위는 출력하고 증가를 한다

java는 참, 거짓을 항상 boolean형으로 해야됨

산술과 관계연산자 중에는 산술이 먼저 -> ex)연산 먼저하고 && 등

논리 연산자에서 A&&B와 B&&A는 서로 다른것이며, A || B 와 B || A가 서로 다른 것임 
ex) A가 참이면 B계산을 하고 결과값을 출력하겠지만, A가 거짓인 경우 B계산을 하지 않고 결과값을 거짓이라고 출력한다
ex) A || B의 경우에는 A가 거짓인 경우에는 B를 계산하지만 A가 참인 경우에는 B를 계산할 필요없이 결과값이 참이라 B를 계산 안함
-> 결과적으로 A,B순서에 따라 연산이 달라지기 때문에 코드 작성할때 주의할 것
