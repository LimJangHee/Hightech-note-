기본자료형 : 8개
클래스의 각 이름들이 참조자료형에 해당 -> 무한함
-> 그래서 참조자료형은 대문자로 시작함
ex)String -> java.lang이라는 패키지에 포함되어 있음

참조자료형의 참조변수는 자기자신의 값을 가지는것이 아니고
해당값은 다른데 저장되어 있고 그 위치(주소값)를 가져오기만 하는 것
그래서 실제 메모리 공간은 2개의 공간이 만들어져야됨
ex)Hello를 저장하는 공간과 이것이 저장되어 있는 주소를 나타내 주는 str(변수) 공간
기본형의 경우 stack메모리에 저장되고, 참조형의 경우 heap메모리에 저장되게 된다


String str = new String("Hello");
참조 자료형은 원래 new를 통해서 만들어야 되는데 -> 2개의 메모리 공간을 할당해줘야 되기 때문에
String의 경우 new를 해줄 필요가 없음
