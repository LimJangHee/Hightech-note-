17.
퀵 정렬 : 레코드의 많은 자료 이동을 없애고 하나의 파일을 부분적으로 나누어 가면서 정렬하는 방식
삽입정렬 : 가장 간단한 정렬 방식, 이미 순서화된 파일에 새로운 하나의 레코드를 순서에 맞게 삽입시켜 정렬
쉘 정렬 : 삽입정렬 확장 개념, 입력파일을 매개변수값으로 서브파일 구성하고 각 서브파일을 삽입 정렬 방식
선택 정렬 : 레코드들 중에서 최소값을 찾아 순서데로 
버블정렬 : 인접한 두개의 레코드 키 값을 비교하여 그 크기에 따라 레코드 위치를 서로 교환하는 정렬 방식
힙 정렬 : 전이진 트리를 이용한 정렬 방식
2-way 합볍 정렬 : 이미 정렬되어 있는 두개의 파일을 한개의 파일로 합병하는 정렬 방식
