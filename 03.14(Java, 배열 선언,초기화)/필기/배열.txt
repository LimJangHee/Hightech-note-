배열 : 사용자가 일일히 변수 선언하는 노가다를 줄여주기위해서
-> 변수 선언을 간편히 하게하기 위해서

연속적인 공간에 고정된 갯수만큼의 같은 타입의 변수를 저장 -> 주소값 하나만 알아도 됨
해당 주소에서 얼마나(몇번째) 떨어져 있는 요소인지를 알려주는 것이 <인덱스> -> 상대적 위치값 : 참조값

배열은 힙메모리에 저장됨 -> new를 써야됨
new를 통해 만들어진 공간들을 객체라고 함 -> 힙 영역에 만들어지는 것들은 객체


알고리즘 : input을 했을때 output이 나오기위한 solution

자료구조 : 같은 타입의 데이터를 효과적으로 저장하고 탐색하는 알고리즘을 모아둔것이 자료구조
- 저장 알고리즘 - 탐색 알고리즘
