메소드 -> ;(문장)들의 집합

class -> 눈에 보이는 사물들을 추상화 시키기 위해서 사용
메소드 -> 객체가 가지고 있는 행동들을 정의하기 위해서 사용
그래서 메소드는 class내부에서만 정의하게 됨

단 class중에서는 추상화의 목적외에도 메소들을 통합하여 사용하기 위해 사용되는데
이럴때의 메소드는 다른 정의를 하게 됨

메소드 정의
void 메소드명() {

} 

-> 메소드명(); 이렇게 불러와서 해당 매소드에 있는 행동을 취할 수 있음
   -> 메인 메소드내에서 메소드를 '호출'

main과 같이 호출을 하는 대상 메소드  '호출자 메소드'
행동을 새롭게 정의해서 불러지는 대상 메소드 '피호출자 메소드'
